Product Requirements Document (PRD): TaskFlow
Project: TaskFlow Landing Page
Version: 1.0
Status: MVP Development

1. Overview
TaskFlow is a minimalist, high-performance landing page for a productivity SaaS application. The goal is to provide a clean, "Apple-esque" marketing page that highlights features and converts visitors into newsletter subscribers.

2. Core Objectives
Conversion: Drive email sign-ups via a prominent CTA (Call to Action).

Speed: Maintain a Lighthouse score of 90+ (achieved through Next.js SSR).

Aesthetics: Use a dark-mode-first, minimalist design language.

3. Key Features
Hero Section: Value proposition statement + Primary CTA button.

Feature Grid: A 3-column layout showcasing key benefits (Real-time Sync, Team Analytics, Smart Reminders).

Footer: Essential navigation and social links.

4. Technical Stack
Framework: Next.js (App Router)

Styling: Tailwind CSS

Icons: Lucide React

Animation: Framer Motion (for smooth hero entrance)

5. UI/UX Requirements
Typography: Clean sans-serif (Inter).

Color Palette: Slate-900 (background), White (text), Indigo-500 (primary accent).

Responsiveness: Fully mobile-first design.
